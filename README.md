# 教学视频转HTML图文学习文档工具

**把课程视频自动变成HTML图文学习工具。**

---

## 这是什么？

你手里有一段视频，里面讲了一些操作步骤。比如：

- 一个编程教程，讲师边写代码边讲解
- 一个软件使用演示，展示怎么完成某个任务
- 一个你自己录的屏幕，记录了某个流程
- 一节在线课程，你想整理成笔记方便复习

你想把视频里的内容整理成一份**图文文档**。这样以后：
- 不用拖着进度条找某个步骤
- 代码可以直接复制粘贴
- 截图可以对照着做
- 可以搜索关键词快速定位

**这个工具可以帮你自动把视频变成图文文档。**

1. 把视频按内容做成切片
2. 每段提取出操作步骤（Step 1、Step 2...）
3. 自动抓取关键画面截图
4. 识别视频里的代码，转成可复制的代码块
5. 输出一份 `.html` 文件，你可以直接打开看，或拖入 AI 知识库

**支持 5 种视觉风格**：抹茶绿 / 极简黑白 / 技术深色 / Notion 风 / 杂志排版

---

## 谁适合用？

| 人群 | 场景 |
|------|------|
| **学习者** | 把课程视频整理成笔记，方便复习和检索 |
| **技术讲师** | 编程/运维/设计课程，给学员发图文版教材 |
| **企业培训** | 内部工具使用教程，新人入职手册 |
| **知识博主** | 把视频内容沉淀为可检索的文档 |

简单说：**只要你手里有视频，想把它变成文字+图片的文档，就能用。**

---

## 需要准备什么？

**必须：**
- 一个 `.mp4` 视频（你的课程录像）
- 一个 AI 模型的 API Key（Kimi / DeepSeek / OpenAI 三选一，用于理解视频内容）

**可选：**
- 字幕文件（`.srt` / `.vtt` / `.txt`）—— 有字幕更准更快；没有的话系统会自动语音转录
- PPT 原稿（`.pptx` / `.pdf`）—— 帮助 AI 更好识别代码和术语

## 怎么用？

### 方式一：有 Mac 电脑（最简单）

1. **下载 DMG 安装包**：

   👉 [点击下载 Course.SOP.Automator-0.1.0-arm64.dmg](https://github.com/coding-with-yiqi/course-sop-automator/releases/download/v0.1.0/Course.SOP.Automator-0.1.0-arm64.dmg)

2. 双击安装，像普通 App 一样打开

   > ⚠️ **macOS 安全提示**：第一次打开时，系统可能会提示「无法打开，因为无法验证开发者」。这是正常的。请去 **系统设置 → 隐私与安全性** 里找到「已阻止使用 Course SOP Automator」，点击 **仍要打开** 即可。
3. 在「设置」页填入你的 API Key（三选一：Kimi / DeepSeek / OpenAI）
4. 回到工作台，拖入视频，点开始
5. 等几分钟，去编辑页微调，导出 HTML

### 方式二：没有 Mac / 想自己部署

**用 AI 编程助手（如 Claude Code、Cursor、Codex）帮你一键搞定。**

你不需要懂代码。只要安装一个 AI 助手，然后把下面的指令发给它：

```
请帮我从 GitHub 下载并运行这个项目：
https://github.com/coding-with-yiqi/course-sop-automator

我需要你：
1. git clone 下来
2. npm install 安装依赖
3. 帮我申请一个 Kimi / DeepSeek / OpenAI 的 API Key
4. 把 Key 写进 .env 文件
5. npm run dev 启动
6. 告诉我访问地址
```

AI 助手会一步一步帮你完成。

**API Key 怎么拿？**（如果 AI 助手不会，你自己来）
- Kimi：https://kimi.com → 开发者中心 → 创建 API Key
- DeepSeek：https://platform.deepseek.com → API 开放平台
- OpenAI：https://platform.openai.com → API keys

## 使用流程

```
上传视频 → AI 自动处理（3-10 分钟）→ 编辑微调 → 导出 HTML
```

**编辑页可以做什么：**
- 修改步骤标题和内容
- 替换不清晰的截图
- 修正代码块里的错误
- 调整详细程度（给新手看 vs 给老手看）

**导出后得到什么：**
- 一个 `.html` 文件
- 打开就是完整的图文文档
- 截图全部内嵌，离线也能看
- 代码可以直接复制

## 常见问题

**Q：没有字幕怎么办？**
A：不用字幕也能跑。系统会自动语音转录（首次需要下载约 190MB 的语音模型，只需一次）。但上传人工校对过的字幕，结果会更准确。

**Q：视频很长（1 小时+）可以吗？**
A：可以。系统会自动按内容切成 ≤25 分钟的段，逐段处理，不会溢出。

**Q：输出的 HTML 能放进飞书/Notion/语雀吗？**
A：目前直接导出 HTML。多平台同步功能正在开发中（M7）。

**Q：我的 API Key 安全吗？**
A：Key 只保存在你本地电脑的数据库里，不会上传到任何服务器。
